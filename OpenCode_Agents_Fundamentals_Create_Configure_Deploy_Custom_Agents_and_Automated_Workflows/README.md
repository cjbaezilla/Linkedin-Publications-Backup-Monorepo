# OpenCode Agents Fundamentals: Create, Configure, Deploy Custom Agents and Automated Workflows

![Cover](./1.jpg)

This folder contains a comprehensive guide to creating, configuring, and deploying custom agents and automated workflows with OpenCode. The guide covers the fundamentals of OpenCode's agent system, including understanding models vs. agents, working with built-in agents like build and plan, creating custom agents using both JSON and Markdown configuration formats, and leveraging specialized subagents like explore and general. You'll learn how to build a team of specialized AI assistants tailored to your development workflow, from security auditing and documentation to code review and performance optimization.
