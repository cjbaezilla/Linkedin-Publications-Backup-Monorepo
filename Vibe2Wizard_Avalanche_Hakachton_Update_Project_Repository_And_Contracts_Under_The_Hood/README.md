# Vibe2Wizard Avalanche Hakachton Update: Project Repository And Contracts Under The Hood

![Cover](./1.png)

This folder contains a comprehensive technical deep-dive into the Vibe2Wizard project built during the Avalanche Build Games Hackathon. The guide explores a complete Web3 learning platform featuring soulbound NFT passports, gamified progression systems, and smart contract architecture. Starting with the core problem of Web3 credentialing, you'll learn about the Wizard Passport implementation with binary search level calculations, the UserRegistration profile system, and the technical decisions behind soulbound tokens. The article then dives into the frontend architecture built with Next.js, covering custom Wagmi hooks for blockchain interaction, React component patterns, wallet integration with RainbowKit, and a nine-step onboarding flow. Each section includes practical code snippets in Solidity and TypeScript alongside natural explanations of concepts like gas-efficient algorithms, on-chain metadata generation, state management with React Query, and responsive UI design with Tailwind CSS and Framer Motion. Whether you're a developer interested in smart contracts, React architecture, or Web3 UX design, this guide provides real-world implementation details from a production-quality hackathon project deployed on Avalanche Fuji testnet.
