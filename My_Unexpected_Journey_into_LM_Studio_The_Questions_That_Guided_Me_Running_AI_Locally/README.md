# Solidity by Example: A Comprehensive Guide with Working Code for Production Smart Contracts

![Cover](./1.jpg)

This folder contains a comprehensive guide to running AI locally on your own computer using LM Studio. Starting from the basics, you'll learn why local AI matters for speed, cost, privacy, and independence. The guide covers understanding what LM Studio actually is, preparing your computer with the right hardware specs, installing the software, and exploring the model library. Each section explains practical concepts like model parameters, thinking capabilities, inference, and temperature settings in simple terms. Whether you've never written a line of code or just want to explore what local AI can do for your daily tasks, this guide takes you step-by-step through the process of setting up your own AI assistant without relying on cloud services or subscription fees.
